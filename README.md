CloudRed (Beta)
===========================

A simple and powerful filesharing app written in Node JS

![alt text](http://i.imgur.com/LBsoeJF.png "CloudRed")

### Install
* Download/Clone this repo
* Run `npm install`
* Run `gulp` and navigate to `http://localhost:3030`
